CLIP Embedding Generator

Generates real AI visual embeddings for every catalog image using your Hugging Face CLIP model.
Run this once whenever you add new parts — then replace data/parts.json on GitHub.

Step 1

Enter your Space URL

Paste your Hugging Face Space URL below. It should be running (green dot on HF).

Step 2

Generate

Click Generate. Each part image is sent to CLIP which returns a 512-number embedding vector.

Step 3

Download & upload

Download the new parts.json and replace it on GitHub. Visual matching is now fully powered by CLIP.

✓ Using real CLIP embeddings (OpenAI ViT-B/32) This is the same model used in professional visual search systems. Results work across different angles, lighting conditions, and photo quality — unlike pixel-based matching.
