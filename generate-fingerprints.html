Atlantic Windows Fingerprint Generator

Image Fingerprint Generator

Run this tool once to generate visual fingerprints for all your catalog images.
The fingerprints enable free, offline image matching — no AI API required.

Step 1

Open this file

Open this HTML file from inside your GitHub repo folder (double-click it). Must be opened from the same folder as your data/parts.json.

Step 2

Generate

Click Generate Fingerprints. The tool loads every part image, analyses colours and patterns, and computes a fingerprint for each one.

Step 3

Save & upload

Copy the output JSON and replace your data/parts.json on GitHub. That's it — matching is live.

⚠ Must be run via a local server Browsers block loading images from local files due to security rules. The easiest way: in VS Code, right-click generate-fingerprints.html → "Open with Live Server". Or run npx serve . in the folder and open localhost:3000/generate-fingerprints.html.

Alternatively, upload this file temporarily to your GitHub repo and open it via the GitHub Pages URL.
